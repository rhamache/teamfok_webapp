Installation Tutorial

This is a online image gallery that allows registered users to upload or view images. Oracle database is required for this web app.

Here are the necessary componets to get you up and running:

  • Setup the lastest version of Tomcat following this link. 
    • http://www.ntu.edu.sg/home/ehchua/programming/howto/Tomcat_HowTo.html
  • Download our service .zip file (proj1.zip) .
  • Unzip or extract the folder inside with a decompression tool (e.g. 7-zip or WinRAR) and place it somewhere you will remember.
  • Place the said folder (proj1) in webapps directory. 
  • In proj1/Setup you will find an SQL script (OurSetup.sql) that you must run on the oracle database. (Note: This combines the provided setup_2014.sql with our own modifications).
    • Option: we have provided you with a java program located in Setup/Populate.java that will populate the web app with test data. Compile and run this program to populate.
    • The Populate.java contains our own testing database credentials. Change the string values in the "getConnected()" method to your own oracle credentials.
  • Open a terimal and start Tomcat.
  • Open a web broswer. The webapp URL is your usual Tomcat URL with "/proj1" appended at the end.
